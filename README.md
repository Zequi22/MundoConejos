# MundoConejos 🐰

¡Bienvenido a MundoConejos! Este proyecto es una página informativa sobre el mundo de los conejos.

## 🚀 Visualización Local

Para visualizar este proyecto en tu máquina local, sigue estos pasos:

1. **Instalar dependencias**:
   Abre una terminal en la carpeta del proyecto y ejecuta:
   ```bash
   pnpm install
   ```
   *(Si no tienes pnpm, puedes usar `npm install`)*

2. **Iniciar el servidor de desarrollo**:
   ```bash
   npm run dev
   ```
